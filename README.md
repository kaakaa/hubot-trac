# hubot-hubot-trac

hubot trac

See [`src/hubot-trac.coffee`](src/hubot-trac.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-hubot-trac --save`

Then add **hubot-hubot-trac** to your `external-scripts.json`:

```json
["hubot-hubot-trac"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
